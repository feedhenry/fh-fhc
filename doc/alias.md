fhc-alias(1) -- FeedHenry Actions
===============================
## SYNOPSIS

    fhc alias <alias>=<app-id>
        
## DESCRIPTION

This is a utility command to set an alias for the 24 character appid 

### app id

Your application id.

### alias

Simple name that is easy to remember


## EXAMPLES
    fhc alias myapp=2ifjg-dsf343hjlo97dhnvh5
