fhc-target(1) -- Get/Set the FeedHenry Target URL
=================================================

## SYNOPSIS

    fhc target
    fhc target <feedhenry-url>
    fhc target <feedhenry-url> <user> <password>
    
## DESCRIPTION

This command will get or set the FeedHenry Target URL.

## EXAMPLES
    fhc target https://apps.feedhenry.com
