fhc-embed(1) -- Build your Application for Embed Target
=======================================================

## SYNOPSIS

    fhc embed <app-id> [dev|live]
    
## DESCRIPTION

This command can be used to build an 'Embed' version of your Application.

## Examples 

    fhc embed mfLkParVTDcr80-uEk8OhEfT --live
