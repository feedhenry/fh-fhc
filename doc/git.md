fhc-git(1) -- FeedHenry App Git Actions
=======================================

## SYNOPSIS

    fhc git [props] <app-id>
    fhc git set <app-id> <git-prop-name> <git-prop-value>
    fhc git pull <app-id>
    
## DESCRIPTION

This command can be used to perform your FeedHenry Git actions on your App.
